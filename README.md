# Solo Builder's Toolkit

Opinionated Claude Code skills for solo AI builders. Each bundle tackles one problem.

Weekly commentary and release notes: [@benjconnor_](https://x.com/benjconnor_) on X.

## What this is

Not a course. Not a marketplace. Not a newsletter. A toolkit — specific tools for specific jobs.

Each skill is a Claude Code slash command. Install the ones you want, invoke them when you need them. New bundle ships every Monday for 7 weeks.

## Install

1. Clone this repo anywhere on your machine.
2. Copy or symlink the skills you want into your Claude Code skills directory (`~/.claude/skills/`). Each skill is a self-contained subdirectory.
3. Restart Claude Code.
4. Invoke via slash commands — `/market-pain`, `/validate-opportunity`, etc.

## Week 1 — Before you build

Four skills shipping Mon 2026-05-04 through Fri 2026-05-08:

- **`market-pain`** — 4-phase research pipeline. Complaint mining → urgency scoring → economic viability → gap analysis. Validates whether a vertical is worth building for.
- **`product-strategy`** — 12-phase gated workflow. Turns a validated pain point into a concrete build plan: platform integration, tech feasibility, service architecture, pricing, resource manifest, 90-day build, kill criteria.
- **`ai-service-designer`** — 8-mode designer. Packages AI services into scoped, priced deliverables with SOPs and first-customer go-to-market.
- **`/validate-opportunity`** (capstone) — Chains the three above. Give it an industry + geography + raw idea. Get back a consolidated GO / KILL / CONDITIONAL verdict with full evidence trail.
